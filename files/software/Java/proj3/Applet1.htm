This page is the finished applet for Project 3 (of 4) in my Java course. It is a game for little children to help them learn their colors and shapes.

First Round:

A 'beep' is sounded when the mouse moves over each shape until the mouse is moved over the yellow circle. The program then proceeds to the second round.

 

Second Round:

A 'beep' is sounded when the mouse moves over each shape until it moves over the red square. The program then proceeds to the third level.

 

Third Round:

This round does not end. Each circle is randomly moved to a new location within the panel when the mouse moves over it.

 

The purpose of this program was to familiarize myself with the creation and design of a GUI as well as introduction to containers, randomization, and user I/O.
